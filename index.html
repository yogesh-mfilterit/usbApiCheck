Device Type Detector — Port / External-Device Signals Only

Uses only external-device and port-related browser APIs: WebUSB, WebHID, Web Serial, already-paired USB peripheral classes, and external-monitor detection. No user-agent, touch, motion, or screen-size checks.

Asymmetry to know: WebHID and Web Serial are desktop-Chrome-only APIs (never on Android Chrome or iOS), so their presence is a strong desktop tell. But there's no port API that's phone-exclusive — Android Chrome and desktop Chrome both expose WebUSB. So this detector can confidently say "desktop" but can only say "uncertain," not a confident "phone," when desktop-only APIs are missing.
